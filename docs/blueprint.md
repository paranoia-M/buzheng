# **App Name**: 天盾风控

## Core Features:

- 智能规则引擎: 可配置化规则引擎：根据不同业务场景，配置风险识别规则，规则即工具，使用大型语言模型确定何时及如何结合不同信息来完成目标。
- 风险等级展示: 风险等级可视化：以不同颜色和标签直观展示风险等级，支持点击查看详细风险报告。
- 风险总览仪表盘: 仪表盘总览：展示核心风险指标，如高风险用户占比、风险事件数量等。
- 自定义报表生成: 自定义报表：允许用户自定义风险报表，并导出为多种格式。
- 风险事件筛选: 多维度筛选：支持按时间、用户属性、行为类型等条件筛选风险事件。
- 实时告警通知: 实时告警：当检测到高风险行为时，立即发出告警通知。
- 模拟数据支撑: 模拟数据生成：使用模拟数据进行系统演示和测试。

## Style Guidelines:

- 主色：深蓝色 (#2E3192)，象征信任、安全与专业。
- 背景色：浅灰色 (#F2F4F7)，营造干净、现代的界面。
- 强调色：亮橙色 (#FF8C00)，用于突出重要操作和信息。
- 正文及标题字体：'Inter'，无衬线字体，现代、简洁、易读。
- 使用简洁、一致的线性图标，强调信息的可读性。
- 采用清晰的栅格系统，保证页面元素的整齐排列。
- 使用轻微的过渡动画，提升交互体验。
